====================================================================
4. CODE STRUCTURE AND PURPOSE
====================================================================

4.1 Repository structure

- dynamic_nav/
  Main project package. Contains environment, models, PPO logic,
  metrics, oracle planning, reporting helpers, and qualitative tools.

- configs/
  Frozen evaluation suite and locked experiment matrix.

- outputs/
  Generated artifacts: training logs, evaluation summaries, figures,
  report assets, and qualitative exports.

- reports/
  Human-facing documentation and generated project writing drafts.

- tests/
  Automated validation for the pipeline and key components.


4.2 Key implementation files

- dynamic_nav/env.py
  Environment definition and state transition logic.

- dynamic_nav/oracle.py
  Time-expanded A* planner used as oracle reference.

- dynamic_nav/models/mlp.py
  Flat baseline policy.

- dynamic_nav/models/gnn.py
  Graph-based relational baseline policy.

- dynamic_nav/models/predictive.py
  Predictive policy with latent rollout and auxiliary loss.

- dynamic_nav/ppo.py
  PPO trainer and training-history generation.

- dynamic_nav/qualitative.py
  Rollout replay, failure classification, qualitative case export.

- train.py
  Unified model training entrypoint.

- eval.py
  Unified evaluation entrypoint, with optional case export.

- run_experiments.py
  Formal run orchestrator for main / ablation / stability runs.

- aggregate_results.py
  Converts raw run outputs into report-grade tables and dictionaries.

- make_figures.py
  Produces comparison figures and training-curve assets.

- generate_report_assets.py
  Generates editable report and presentation drafts.

- export_rollouts.py
  Exports a specific episode as a qualitative case bundle.


4.3 Important artifact directories

- outputs/eval/
  Evaluation summaries and per-episode CSV files

- outputs/train_logs/
  Training summaries and training-history CSV files

- outputs/report_assets/
  Pivot tables, report numbers, narrative notes, failure summaries

- outputs/figures/
  Final chart assets and copied qualitative figures

- outputs/qualitative/
  Rollout JSON / CSV / TXT / SVG case bundles
